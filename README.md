# ⏰ Alarme Online

Esse projeto é uma reconstrução com base no antigo design de um site de alarme online que utilizei muitos anos durante a faculdade. Prática de técnicas em programação utilizando HTML, CSS e JavaScript.

## 📌 Sobre o Projeto

O objetivo deste projeto é exibir um relógio com tempo real e permitir que o usuário agende um horário específico para disparar um alarme sonoro diretamente pelo navegador.
---

## 🚀 Funcionalidades

- Interface simples;
- Exibir relógio em tempo real;
- Definir horário do alarme;
- Agendar o horário do alarme;
- Disparar o alarme com áudio selecionado;
- Desligar o alarme;
---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura da página |
| CSS3 | Estilização |
| JavaScript | Lógica do alarme |
